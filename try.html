History of HTML

HTML, or HyperText Markup Language, was first created by Tim Berners-Lee in 1991 while working at CERN. It was designed to allow scientists to share documents and link between them using hyperlinks.

The first version, HTML 1.0, was very basic. It allowed for headings, paragraphs, lists, and links. Over time, new features were added:

  • HTML 2.0 (1995) Standardized version with basic form elements
  • HTML 3.2 (1997) Introduced style support and scripting (JavaScript)
  • HTML 4.01 (1999)Improved structure and support for CSS
  • HTML5 (2014 - present) Major update with multimedia, semantic elements, and better performance

Today, HTML5 is the core language of the web, supported by all modern browsers and used to create everything from simple pages to complex web applications.
