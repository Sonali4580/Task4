Introduction to HTML

Learn how HTML and CSS work together to build web pages

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It defines the structure and content of a webpage using tags.

Basic HTML Tags

  • <html>Root element
  • <head>ontains meta information like title
  • <body> Contains the visible page content
  • <h1> to <h6> Headings
  • <p> Paragraph
  • <a> Hyperlink
  • <img> Image
  • <ul>, <li> Lists

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

How CSS Helps

CSS (Cascading Style Sheets) is used to make your HTML pages look beautiful. With CSS, you can:

  • Change colors and fonts
  • Control layout and spacing
  • Make pages responsive for all screen sizes
  • Add effects like shadows, borders, and animations

Example:

<style>
  h1 {
    color: blue;
    text-align: center;
  }
  p {
    font-size: 18px;
    color: gray;
  }
</style>
