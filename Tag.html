Basic HTML Tags and Examples

<h1> to <h6> — Headings

Headings define the titles or subtitles on a web page. <h1> is the largest, and <h6> is the smallest.

<h1>Main Heading</h1>
<h2>Subheading</h2>

<p> — Paragraph

The <p> tag defines a paragraph of text.

<p>This is a paragraph of text.</p>

<a> — Anchor (Link)

The <a> tag defines a hyperlink to another page or website.

<a href="https://example.com">Visit Example</a>

<img> — Image

The <img> tag is used to display images. It requires a src and an alt attribute.

<img src="image.jpg" alt="Sample Image" />

<ul>, <ol>, <li> — Lists

<ul> creates an unordered (bulleted) list, and <ol> creates an ordered (numbered) list. Items go inside <li>.

<ul>
  <li>Item One</li>
  <li>Item Two</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

<br> — Line Break

The <br> tag inserts a single line break. It is a self-closing tag.

Line one<br>
Line two

<strong> and <em> — Emphasis

<strong> makes text bold. <em> italicizes text.

<p>This is <strong>important</strong>.</p>
<p>This is <em>emphasized</em>.</p>
