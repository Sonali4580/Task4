Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create webpages. It structures the content and elements of a webpage like headings, paragraphs, links, and images.

Basic Structure of an HTML Document

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>Welcome to HTML.</p>
  </body>
</html>

Common HTML Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraphs
  • <a>: Links
  • <img>: Images
  • <ul>, <ol>, <li>: Lists
